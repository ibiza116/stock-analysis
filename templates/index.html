📚 高度テクニカル指標の見方・使い方

🎯 ボリンジャーバンド

概要: 移動平均線を中心とした価格の変動幅を示すバンド

構成:

  • 中央線：20日移動平均線
  • 上限・下限：標準偏差±2σ

投資判断:

  • 🔴 上限タッチ → 売りシグナル（買われすぎ）
  • 🟢 下限タッチ → 買いシグナル（売られすぎ）
  • 🟡 バンド幅縮小 → 大きな値動き前の前兆

📊 MACD

概要: 移動平均収束拡散指標。トレンドの強さと転換点を判定

構成:

  • MACD線：12日EMA - 26日EMA
  • シグナル線：MACD線の9日EMA
  • ヒストグラム：MACD線 - シグナル線

投資判断:

  • 🟢 ゴールデンクロス → 買いシグナル
  • 🔴 デッドクロス → 売りシグナル
  • 📈 ヒストグラム拡大 → トレンド強化

⚡ ストキャスティクス

概要: 一定期間の価格レンジ内での現在価格の位置を%で表示

構成:

  • %K線：高値・安値レンジでの現在価格位置
  • %D線：%K線の3日平滑移動平均

投資判断:

  • 🔴 80%以上 → 買われすぎ（売り検討）
  • 🟢 20%以下 → 売られすぎ（買い検討）
  • ✨ %K > %D → 上昇の勢い

📈 出来高分析

概要: 価格変動と出来高の関係を分析。トレンドの確実性を判定

構成:

  • 出来高：実際の取引量
  • 出来高移動平均：20日間の平均出来高

投資判断:

  • 🔥 2倍以上 → 異常出来高（重要な動き）
  • 📊 1.5倍以上 → 高出来高（注目）
  • 💡 上昇+高出来高 → 上昇トレンド信頼性高
