📚 高度テクニカル指標の見方・使い方

🎯 ボリンジャーバンド

概要: 移動平均線を中心とした価格の変動幅を示すバンド

構成:

  • 中央線：20日移動平均線
  • 上限・下限：標準偏差±2σ

投資判断:

  • 🔴 上限タッチ → 売りシグナル（買われすぎ）
  • 🟢 下限タッチ → 買いシグナル（売られすぎ）
  • 🟡 バンド幅縮小 → 大きな値動き前の前兆

📊 MACD

概要: 移動平均収束拡散指標。トレンドの強さと転換点を判定

構成:

  • MACD線：12日EMA - 26日EMA
  • シグナル線：MACD線の9日EMA
  • ヒストグラム：MACD線 - シグナル線

投資判断:

  • 🟢 ゴールデンクロス → 買いシグナル
  • 🔴 デッドクロス → 売りシグナル
  • 📈 ヒストグラム拡大 → トレンド強化

⚡ ストキャスティクス

概要: 一定期間の価格レンジ内での現在価格の位置を%で表示

構成:

  • %K線：高値・安値レンジでの現在価格位置
  • %D線：%K線の3日平滑移動平均

投資判断:

  • 🔴 80%以上 → 買われすぎ（売り検討）
  • 🟢 20%以下 → 売られすぎ（買い検討）
  • ✨ %K > %D → 上昇の勢い

📈 出来高分析

概要: 価格変動と出来高の関係を分析。トレンドの確実性を判定

構成:

  • 出来高：実際の取引量
  • 出来高移動平均：20日間の平均出来高

投資判断:

  • 🔥 2倍以上 → 異常出来高（重要な動き）
  • 📊 1.5倍以上 → 高出来高（注目）
  • 💡 上昇+高出来高 → 上昇トレンド信頼性高

🔄 バックテストとは？

バックテストは、過去の株価データを使って投資戦略の有効性を検証するシミュレーション機能です。

📊 何ができる？

  • 過去の相場で戦略が利益を出せたか確認
  • リスクとリターンの関係を数値で把握
  • 複数の投資戦略を比較検討
  • 実際の投資前にリスクを評価

⚠️ 注意点

  • 過去の結果は将来を保証しません
  • 手数料・税金は考慮されていません
  • 流動性リスクは含まれていません
  • 参考情報として活用してください

📈 使い方の流れ

  1. 検証したい期間を設定
  2. 投資戦略を選択
  3. 初期資金を入力
  4. 実行ボタンでシミュレーション開始
  5. 結果を分析して戦略を評価

⚙️ バックテスト設定

📅 検証期間

過去のどの期間で戦略をテストするかを設定します

🎯 投資戦略

どの売買ルールでテストするかを選択します

複合戦略: RSI、移動平均線、MACD、ボリンジャーバンドを組み合わせた総合的な判定を行います。複数の指標を使うことでより精度の高い売買タイミングを狙います。（初心者におすすめ）

💰 初期資金

投資をスタートする時の資金額を設定します

📊 バックテスト結果

📈 総リターン

投資期間全体での利益率。プラスなら利益、マイナスなら損失を表します。年利換算して他の投資商品と比較しましょう。

🎯 勝率

全取引のうち利益を出した取引の割合。高いほど安定していますが、50%以上あれば良好です。

📉 最大ドローダウン

最大の損失幅。投資で覚悟すべき最大の下落を示します。10%以下なら良好、20%超は要注意です。

⚖️ シャープレシオ

リスクに対するリターンの効率性。1.0以上で良好、2.0以上で優秀な戦略と言えます。

🏪 バイ&ホールド比較

単純に買って持ち続けた場合との比較。プラスなら戦略に価値があることを示します。

📋 総合判定

優秀: 全指標が良好　良好: 多くの指標が良好　要改善: リスクが高いか収益性が低い
